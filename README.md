# 🛒 E-commerce Web Application

A full-stack e-commerce platform for browsing products, filtering items, and managing a complete shopping experience.

---

## 🚀 Live Demo

https://xperia-e-commerce.vercel.app/

---

## 🛠️ Tech Stack

### 🎨 Frontend

* ⚛️ Next.js
* ⚛️ React
* 🎨 Tailwind CSS
* 🧠 Context API

### 🔧 Backend

* 🟢 Node.js
* 🚀 Express.js

### 🗄️ Database

* 🍃 MongoDB

---

## ✨ Features

* 🔐 Authentication System (Login / Register)
* 🛍️ Product Listing & Details Pages
* 🔍 Advanced Filtering (Brand / Year / Specs)
* 🛒 Shopping Cart System
* 💳 Checkout Flow
* 📱 Responsive & Clean UI
* ♻️ Reusable Components Architecture
